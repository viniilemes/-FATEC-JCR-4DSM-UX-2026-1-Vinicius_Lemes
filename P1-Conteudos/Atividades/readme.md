# Atividades - P1

## 📋 Descrição

Conjunto de atividades práticas desenvolvidas ao longo do primeiro período, focadas em aplicar conceitos de Experiência do Usuário (UX) e Engenharia de Usabilidade.

## 🎯 Objetivo

Realizar atividades estruturadas para consolidar o aprendizado sobre:
- Design de interfaces
- Pesquisa com usuários
- Prototipagem
- Testes de usabilidade

## 📂 Estrutura

- **Atividade-Aula01**: Atividade introdutória de UX
- **Atividade-Aula02**: Mapa de expectativas e exercícios de design
- **Atividade-Aula03**: Descrição de atividade e respostas
- **Atividade-Aula04**: Fluxo de usuário e tabelas de apoio
- **Atividade-Aula05 até Aula08**: Atividades progressivas do curso
