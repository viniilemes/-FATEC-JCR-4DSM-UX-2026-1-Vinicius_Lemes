# Atividade - Aula 02

## 📋 Descrição

Atividade de aprofundamento em mapping e exercícios práticos de design, incluindo mapa de expectativas e modelagem de fluxos.

## 🎯 Objetivo

Desenvolver habilidades de empatia com o usuário através de técnicas de mapa de expectativas e exercícios estruturados de design.

## 📁 Arquivos

- **EU-Atividade-Aula02-Descrição.pdf**: Descrição e instruções da atividade
- **EU-Modelo-Exercício1.docx**: Modelo para exercício 1
- **EU-Modelo-Exercício2.docx**: Modelo para exercício 2
- **EU-Modelo-Mapa de expectativas.docx**: Template de mapa de expectativas
- **Exercicio1-resposta.docx**: Resposta do exercício 1
- **exercicio1.pdf**: Versão PDF do exercício 1
- **mapa-expectativas.pdf**: Mapa de expectativas preenchido

## 📝 Conteúdo Abordado

- Mapa de expectativas
- Exercícios de design
- Comunicação visual
- Técnicas de empatia
