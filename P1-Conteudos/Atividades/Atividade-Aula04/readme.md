# Atividade - Aula 04

## 📋 Descrição

Atividade focada em fluxo de usuário e tabelas de apoio, com exemplos práticos de representação visual de processos e interações.

## 🎯 Objetivo

Desenvolver a capacidade de representar fluxos de usuário através de diagramas e tabelas de apoio estruturadas.

## 📁 Arquivos

- **EU-Atividade-Aula04-Descrição.pdf**: Descrição completa da atividade
- **EU-01-Folha de Apoio em Tabela-ExemploPreenchido.pdf**: Exemplo preenchido de tabela
- **EU-02-Modelo Vazio de Fluxo de Usuário.html**: Template de fluxo em HTML
- **EU-03-Folha de Apoio em Tabela.docx**: Documento de tabela de apoio
- **fluxogramaFrota.html**: Exemplo de fluxograma interativo
- **tabelaApoiaoFluxoUsuário.MD**: Tabela de apoio em Markdown

## 📝 Conteúdo Abordado

- Modelagem de fluxo de usuário
- Tabelas de apoio e documentação
- Fluxogramas interativos
- Representação visual de processos
