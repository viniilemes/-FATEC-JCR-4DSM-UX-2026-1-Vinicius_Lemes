## 📌 Projeto ABP (Aprendizado Baseado em Problemas)

O projeto principal aborda um **problema real** através de metodologia ABP, integrando:

1. **Identificação do Problema** - Análise e compreensão do contexto
2. **Pesquisa com Usuários** - Entrevistas e validação de hipóteses
3. **Design da Solução** - Wireframes, protótipos e iterações
4. **Testes de Usabilidade** - Validação com usuários reais
5. **Documentação** - Apresentação dos resultados e aprendizados

📄 Veja [Requisitos-ABP/readme.md](./Requisitos-ABP/readme.md) para mais detalhes.
